# TMXC

Tmux configurations.

## Installation

```
git clone https://github.com/dotcon/tmxc
cd tmxc
ln -s $(pwd)/.tmux.conf ~/.tmux.conf
```

After the above operations, start `tmux`, then press `prefix + I` to install the configured plugins.
